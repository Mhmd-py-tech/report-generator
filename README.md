## What It Does
- Reads CSV sales data
- Calculates summary statistics (totals, averages)
- Generates styled HTML report with:
  - Summary cards showing key metrics
  - Month-by-month data table
  - Professional dashboard layout

## Preview
[View Sample Report](report.html) *(Click "View raw" or download to see full styling)*

## How To Use
1. Add your sales data to `sales_data.csv` (format: month, revenue, customers, deals_closed)
2. Run `python generator.py`
3. Open `report.html` in your browser
